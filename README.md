# FIFA 26 — World Cup 2026 Companion

A retro, teletext-style companion app for the 2026 FIFA World Cup. Follow the
whole tournament from one clean, fast interface.

## What you can do

- ⚽ **Watch live score updates** — follow matches as they happen, with scores that refresh on their own.
- 📊 **Track group standings** — see how every group is shaping up in real time.
- 📅 **Browse the fixtures** — full match schedule with kickoff times in your local timezone.
- 👥 **Explore squads** — line-ups and player lists for every nation.
- 🏟️ **Discover the venues** — stadiums and host cities across the tournament.
- 📺 **Retro teletext vibe** — a nostalgic, distraction-free way to keep up with the action.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
